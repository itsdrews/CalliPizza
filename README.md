# CalliPizza

Essa é uma aplicação ReactJS para o sistema de gerenciamento de uma pizzaria.

Funcionalidades: 
* Login (Autenticação) para Funcionário e Administrador
* CRUD de produtos (Pizzas)

```npm install ```
## Bibliotecas Necessárias:
* yup
* react-hook-forms
* multer
* react-router-dom
* axios
* @mui/icons-material
* react-tooltip
* react-toastify
* express
* react-hookform-resolvers
